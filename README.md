## Exercício 01 ##
Uma loja virtual está desenvolvendo um módulo simples para controle de produtos
cadastrados em seu sistema. Cada produto possui informações essenciais que precisam ser
armazenadas e manipuladas pelo programa.
A equipe de desenvolvimento decidiu utilizar o paradigma de Programação Orientada a
Objetos (POO) para organizar melhor o código e facilitar futuras expansões do sistema. Sua
tarefa é implementar a estrutura básica dessa solução.
A classe produto deve conter obrigatoriamente 3 atributos (nome, valor, quantidade
disponível) e o método construtor para inicializar os atributos. Crie também um método para
aumentar o valor do produto a partir de uma porcentagem informada pelo usuário da
aplicação.
